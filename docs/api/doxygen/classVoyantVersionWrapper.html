Creates a VoyantVersionWrapper from a 32-bit hash in the MMNNPPSC format

The hash has the following bit layout: [31:24] - Major version (8 bits) [23:16] - Minor version (8 bits) [15:8] - Patch version (8 bits) [7:4] - Status (4 bits) [3:0] - Component type (4 bits)

◆ toU32Hash()

uint32_t VoyantVersionWrapper::toU32Hash ( ) const

Converts the version to a 32-bit hash in the format MMNNPPSC where:

  • MM: Major version (8 bits)
  • NN: Minor version (8 bits)
  • PP: Patch version (8 bits)
  • S: Status (4 bits)
  • C: Component type (4 bits)

Returns a uint32_t with the following bit layout: [31:24] - Major version [23:16] - Minor version [15:8] - Patch version [7:4] - Status [3:0] - Component type
